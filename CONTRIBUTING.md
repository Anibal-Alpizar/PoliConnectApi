# Contributing

## Contributing code

1. Clone this repository:

```
git clone https://github.com/kokoaespol/PoliConnectApi
```

2. Code your contribution following the non-existing style guides.

3. Create a PR for us to review your code.

## Opening an issue

Head over to the [issues tab](https://github.com/kokoaespol/PoliConnectApi/issues) and create a new
issue with your suggestion or problem.
